Haloband 全解析

2014-08-25

Haloband到底是什么？

从硬件上来说Haloband就是 3 个 NFC 标签，两个作为触发器，一个作为活标签(自造词)，何为活标签？不需要改写 NFC 标签，却可以改变里面承载的内容。也即是 Haloband 所谓的云服务。

下面我将从各方面来解析Haloband。

触发器

Haloband有 A，B 两个触发器。可以实现 6 个触发动作：A, B, A->B, B->A, A->A, B->B。
每个动作可以触发一个任务。
什么社交签到，切歌，手电筒，SOS 地理位置，拨打电话，导航，抓拍，解锁手机都是基于此。

那么这个是原创的吗？

当然不是。有一个神器玩 NFC 的人大概都知道，Trigger(原名 NFC Task Launcher)。我不知道这是不是最早的，这个我无法考究，但应该是很早。Trigger的触发器种类非常丰富。免费版的有 wifi，蓝牙，NFC。付费版的有电池电量，时间，充电，耳机，日历时间，甚至是地理位置。任务也是非常的丰富。就不展开了，自行下载体验。没有 NFC 也能用哦。

Haloband是云飞网的作品。云飞网也有类似Trigger的作品。可是功能就比较简陋了，叫云飞 NFC。

云飞 NFC 界面

那么Haloband在这个方面有创新的地方吗？

还是有的。不需要写东西到 NFC 标签。利用两个触发器，实现 6 个触发动作。更加适合移动使用。加入了比较 fashion 的东西，如控制飞利浦的 Hue 电灯。加入了IFTTT，可以实现更多有趣的功能。

IFTTT，是一个新生的网络服务平台，通过不同其他平台的条件来决定是否执行下一条命令。即对网络服务通过其他网络服务作出反应作出反映。IFTTT 得名为其口号「if this then that」。

NFC 标签作为触发器是合适的。可以放在移动的场合，就些不伦不类了。
会有误触的问题，而且手环是在动的。标签的位置并不固定。手还得转一下，才能碰到标签。
这看起来是个很二的行为。

云服务

cloud 标签是一个活标签。因为没有Haloband实物，所以软件也不能用。还好在云飞网找到了这样的云服务。
每个标签有一个固定的链接。通过改变云端的内容，来实现不用的功能，都是通过浏览器实现。云飞网提供了 URL，电话号码，名片，文本，HTML，投票六个选项。

其中投票不可用。HTML，文本，电话号码用的技术其实是一模一样，就是加载一个静态的 HTML 文档，电话号码就是加上tel：的标志，来告诉浏览器这是一个电话号码，点击即可拨号，网页非常简陋。URL 就是一个跳转服务，类似于新浪的短地址。名片就是把你填写的内容，生成一个 VCF 格式的名片，下载后，打开会自动导入到手机中。名片功能是亮点，这里点个赞。

草料二维码有提供类似服务。二维码和 NFC 标签其实都是一样的。只是一个把文本内容变成二维码，一个写入到标签而已。

解锁、签到

解锁、签到的核心都是用手环来作为身份 ID。

Haloband可以解锁手机和应用。NFC 可以是可以加密的，所以安全性也有一定的保障。当然这个保障是相对的，还是有办法破解的。
给应用加锁并不需要 ROOT 权限。只要开启一个服务监听栈顶应用程序，如果是加锁的应用，则弹出验证界面。

如果你认为你买了Haloband就可以替代门禁卡了。那你就太天真了。
需要门禁系统的硬件和软件支持才可以的哦。

当然我始终认为，用手环这样的东西。作为身份 ID 并不是一个很好的方案。
人身上有能够识别身份的东西，比如指纹。更安全，也更优雅。

营销

在国外众筹的时候，还挺高大上的。怎么一上国内就 Low 了呢。
各位看官自行对比 kickstarter，淘宝众筹。

各种调侃小米手环，根本不是一个品类好吗？
这样偷换概念，拿一个不是一个品类的东西做对比，真的好吗？
反正我是瞬间感觉被侮辱智商。

不过国人吃这一套，看支持数量就知道了。价格 39 元，相比在国外的售价算是上良心了。不能单纯的以物料成本计算。

结语

创新的东西有，但没有自己的核心技术。让人们知道，手环还可以这么做，虽然我觉得有些鸡肋。Haloband是一个可以玩的东西，但新鲜以后呢，能给用户带来多大的价值，值得思考。

最后还是希望越做越好，给中国人长面。不做Copycat，继续创新。
